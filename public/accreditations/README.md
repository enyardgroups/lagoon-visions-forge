# Accreditation Logos

Place your accreditation logo images in this directory.

## Required Logo Files

Please add the following logo files:

- `iso27001.png` - ISO 27001 Certified logo
- `dun-bradstreet.png` - DUN & Bradstreet logo

## Logo Specifications

- **Format**: PNG (preferred) or SVG
- **Recommended height**: 48px (h-12 in Tailwind)
- **Background**: Transparent background preferred
- **File naming**: Use lowercase with hyphens (as shown above)

## Usage

The logos will be displayed in the footer's accreditations section. If a logo file is missing, a text fallback will be displayed automatically.

## Links

- ISO 27001: Displayed as a badge
- DUN & Bradstreet: Links to https://profiles.dunsregistered.com/TPIN-BAS-004.aspx
