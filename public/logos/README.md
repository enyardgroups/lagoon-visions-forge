# Partner Company Logos

Place your partner company logo images in this directory.

## Required Logo Files

Please add the following logo files:

- `boobazhop.png` - BoobaZhop logo
- `uae-chauffeur.png` - UAE Chauffeur logo
- `first-class-fleet.png` - First Class Fleet logo
- `enyard.png` - Enyard logo

## Logo Specifications

- **Format**: PNG (preferred) or SVG
- **Recommended size**: 200x80 pixels (or similar aspect ratio)
- **Background**: Transparent or white background
- **File naming**: Use lowercase with hyphens (as shown above)

The logos will be displayed in a scrolling marquee on the home page. If a logo file is missing, the company name will be displayed as a fallback.
